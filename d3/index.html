Building Data Visualisations with D3

Make data make sense.
Learn to create interactive data visualisations for the web, using the incredible JavaScript toolkit, D3.

Whether you want to create your own social media analytics, compare latest trends, construct political maps, graphs and charts or visualise your organisation's progress, you can put that data to good use with this hands-on foundation in data visualisation.

3rd July 2013, 9am-5pm at L4RP, Brighton

D3.js is the gold standard of data visualisation and can be used to create an amazing range of charts. In this one day workshop, you'll learn what D3 is, what it can do and how it works. This will include an introduction to SVG, the web standard language for vector graphics.

You'll need to be comfortable with JavaScript for this workshop. (If you want to learn JavaScript, see Coding For Humans).

Taught by

Peter Cook, a web developer who specialises in data visualisation. He enjoys the entire visualisation process, from conceptual design through to implementation.

We want to spread this knowledge far and wide, and we've tried to make this course as affordable as possible. If you are unwaged and cannot afford the course but would greatly benefit it, please contact us about our hardship fund.

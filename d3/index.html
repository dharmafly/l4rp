A one day introduction to building data visualisations, using D3.

You may have heard of D3.js and maybe seen amazing examples of data visualisations built with it. But what is it, what’s it good for and how does it work?

The workshop assumes a basic to intermediate grasp of JavaScript. We'll start right at the beginning with D3, explain how it fits into the data visualisation landscape, how it works and how to build visualisations with it. This will include an introduction to SVG, the web standard language for vector graphics.

By the end of the day, you will understand D3 well enough to be able to build your own data visualisations.

Taught by

Peter Cook, a web developer who specialises in data visualisation. He enjoys the entire visualisation process, from conceptual design through to implementation.

"Code Up"

Coding for non-coders ~ Tues, 24th July, 2012

A one day workshop at the Lab for the Recently Possible, 9am - 5pm.

Coding is a means of expression and creativity. It's a satisfying and potent skill, in an ever-increasingly webbified world.

Whether you have an interest to work in web development or are just a curious bystander of the Web at work or home, then here is a chance to spread your wings and learn to code a little… In a single day… Even if you've never written or seen a line of code before.

Step-by-step

By the end of the day, you'll have learned to write the basics of the languages that run every website, page and application on the Web: HTML is the language that describes the content on a web page, CSS adds styling (e.g. colours, text size and layout) to that content, and JavaScript is used to make the content interactive.

This is a beginners' course. Intermediate and advanced sessions are coming soon.

Prerequisites: basic familiarity with using the Web, basic familiarity with using a computer, and access to a laptop that you can bring along (or contact us if you'd like to borrow one).

In good hands

Hosted by: Tom Parslow, Pete Goodman and Premasagar Rose - all Brighton-based professional web developers and teachers of Web technology.
